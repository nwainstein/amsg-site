# AMS Group Website

Static website for the AMS research group.

## Contents

- `index.html` - homepage
- `research.html` - research overview
- `publications.html` - publications list
- `people.html` - team members
- `news.html` - news items
- `contact.html` - contact page
- `assets/` - CSS, JS, and images

## Development

1. Open `index.html` in a browser.
2. Edit individual HTML/CSS/JS files in the folder.

## GitHub Pages deployment

1. In GitHub repository settings, go to **Pages**.
2. Select branch `main` and folder `/` (root).
3. Save and wait for site URL.
